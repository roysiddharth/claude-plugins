# claude-plugins

Siddharth Roy's Claude Code plugin marketplace.

## Install

```bash
claude plugin marketplace add github.com/roysiddharth/claude-plugins --scope user
```

## Plugins

| Plugin | Install | Description |
|--------|---------|-------------|
| dev | `claude plugin install dev@roysiddharth` | Full AI-assisted dev workflow: plan, spec, issue, implement AFK |
